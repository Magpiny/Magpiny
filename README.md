### 💬 [Malama](https://github.com/Magpiny/malama)
*Native Linux chat client for local LLMs — no cloud, no browser, no compromise*

A desktop client for Ollama that behaves like a native app should: it idles around 20 MB of RAM, streams tokens over a non-blocking Boost.Asio socket, and renders Markdown with its own zero-dependency parser. Sessions persist to SQLite, the model list populates live from your local Ollama instance, and a reasoning-block filter keeps `<think>` output out of the way.

### 📊 [sysinfoviewer](https://github.com/Magpiny/sysinfoviewer)
*A system monitor that doesn't need a terminal*

Everything `htop`, `upower -i`, and `lscpu` show separately, in one lightweight native window: live CPU and memory charts, UPower-grade battery detail (health, cycles, vendor, serial), disk usage, running processes, and installed applications.
